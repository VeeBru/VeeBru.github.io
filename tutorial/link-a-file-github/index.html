permalink + ?raw=true

Mi chiedevo se fosse possibile usare una repository di github come un archivio di materiale ( fotografie, documenti, cose da allegare ad email..)

anche se cliccando sul file nella repository permette di copiare il permalink al file, le mie prove mi hanno fatto scoprire non è possibile usarlo per la condivisione: l’errore è File not Found

spulciando vari forum ho capito che basta aggiungere

?raw=true

alla fine del permalink che ci da Github

https://github.com/nome-utente/repository/blob/id-commit/nome-file.estensione?raw=true

forse è una cosa banale ma io non lo sapevo!
